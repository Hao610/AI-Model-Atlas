# 🧭 AI Model Atlas 快速入门指南

← 返回 [中文首页](../README_zh.md) | [English Version (GETTING_STARTED.md)](GETTING_STARTED.md)

---

欢迎来到 **AI Model Atlas** 仓库！本项目旨在构建一个结构化的大语言模型（LLM）知识与技术地图——打通零代码工具、全栈软件开发与底层核心数学/算法原理之间的断层。

为了帮您高效地探索本仓库，请根据您的背景和目标选择最适合您的阅读路径：

---

## 🗺️ 选择你的路径

### 1. 我是学生或自学者，想寻找一个循序渐进的学习路线图
> **推荐入口**：[学习课程体系 (CURRICULUM_zh.md)](CURRICULUM_zh.md)
* **内容定位**：一套涵盖 4 大阶段、36 个模块的学习路线。从零 AI 基础一步步通关到云端大规模微调与部署。
* **适用场景**：制定系统性的自学计划，顺序学习提示词工程、Dify 工作流开发与 Python API 调用。

### 2. 我是软件工程师或架构师，想构建商用生产级系统
> **推荐入口**：[系统架构文档 (ARCHITECTURE_zh.md)](ARCHITECTURE_zh.md)
* **内容定位**：针对工程落地设计的架构文档，包含生产级 RAG 系统的延迟跑分、Ollama 本地服务故障自愈/容灾机制以及运行状态机。
* **适用场景**：理解工业级 LLM 应用的性能指标、状态流转与容灾设计模式。

### 3. 我是技术极客或学术研究人员，想探究底层技术与算法原理
> **推荐入口**：[极客深潜专题 (DEEP_DIVES_zh.md)](DEEP_DIVES_zh.md)
* **内容定位**：一部 17 章节的技术科普系列，追踪 AI 算法史的演进：从古典机器学习与特征工程，到 Transformer 结构、向量 HNSW 检索、RAG 逻辑、Agent 系统、MoE 架构以及基于强化学习的推理模型。
* **适用场景**：获得直观、可视化的底层数学与工程折中方案解释（例如 Chunk 尺寸权衡、Reranker 重排、推理扩展定律 Inference Scaling Laws）。

### 4. 我只想立刻克隆仓库，跑起来看效果
> **推荐入口**：[中文版首页 - 运行路径选择](../README_zh.md#-运行路径选择)
* **内容定位**：快速安装和上手指南，手把手教你如何配置本地 Ollama 服务、运行交互式命令行以及测试 RAG 知识库。
* **适用场景**：希望在 5 分钟内搭建好本地测试沙盒环境的开发者。

### 5. 我是专注于 AI 安全的安全工程师或 DevSecOps 架构师
> **推荐入口**：[约束与威胁模型 (CONSTRAINT_THREAT_MODEL_zh.md)](CONSTRAINT_THREAT_MODEL_zh.md)
* **内容定位**：长达 36 章的架构蓝图，详细说明如何将传统的应用安全（AppSec）映射到 LLM 时代。涵盖提示词注入、数据外泄、上下文篡改以及如何构建零信任架构边界。
* **适用场景**：安全研究人员、DevSecOps 团队和企业架构师，致力于构建稳健、具备高弹性且安全的 LLM 应用系统。
