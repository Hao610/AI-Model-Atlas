# 计费与 Token 经济学 💳

[[English] (16_cost_and_tokens.md)](16_cost_and_tokens.md) | [中文]

在开发一个 AI 应用时，很多新手或企业管理者最常问的问题就是：*“这玩意儿到底要花我多少钱？”* 不同于传统云服务按照服务器开机时间计费，商业大模型 API 主要是按照 **Token (字词碎片) 的消耗量** 来收费的。

本章将带你算清这笔账，学会科学评估你的项目预算。

---

## 🪙 1. 大模型 API 计费的两个维度

无论是 OpenAI 还是 DeepSeek，其 API 账单都分为两列：

1. **输入 Token 费 (Input/Prompt)**：你发送给模型的字数（包括你的提示词、系统指令、以及通过 RAG 检索出来并塞给模型的参考文档字数）。
2. **输出 Token 费 (Output/Completion)**：大模型打字回复给你的字数。

*注意：输入的单价总是比输出便宜得多。因为大模型生成新字（输出）需要显卡进行大量的逐步计算，消耗更多算力。*

---

## 📈 2. 常见模型计费对比 (每 100 万 Token 单价)

为了让你对价格有一个直观的概念，以下是目前市场上主流大模型 API 的单价对比（以每 100 万 Token 消耗的美元/人民币计费）：

| 模型名称 | 输入单价 (每 1M tokens) | 输出单价 (每 1M tokens) | 性价比区间 |
| :--- | :--- | :--- | :--- |
| **GPT-4o** | \$2.50 (约 18元) | \$10.00 (约 72元) | 🟥 昂贵贵族 |
| **Claude 3.5 Sonnet** | \$3.00 (约 21元) | \$15.00 (约 108元) | 🟥 昂贵贵族 |
| **GPT-4o-mini** | \$0.15 (约 1.1元) | \$0.60 (约 4.3元) | 🟨 亲民平价 |
| **DeepSeek V3 / R1** | \$0.14 (约 1.0元) | \$0.28 (约 2.0元) | 🟩 极致白菜价 |

### 🧮 算力账本：实战算账案例
假设你搭建了一个 RAG 企业智能客服机器人，每天需要自动回复 **1000 个** 客户提问：
* 每次提问由于需要把 FAQ 文档和聊天历史塞给大模型，导致**输入 Token 数达到 1500 个**。
* 大模型平均回复一句简短的话，**输出 Token 数为 200 个**。

如果使用 **GPT-4o** 计费：
$$\text{输入费用：} 1000 \times 1500 \text{ tokens} = 1.5\text{M tokens} \times \$2.50 = \$3.75$$
$$\text{输出费用：} 1000 \times 200 \text{ tokens} = 0.2\text{M tokens} \times \$10.00 = \$2.00$$
$$\text{单日总开销} = \$5.75 \text{ (每月约合人民币 1200 元)}$$

如果使用 **DeepSeek V3** 计费：
$$\text{输入费用：} 1.5\text{M tokens} \times \$0.14 = \$0.21$$
$$\text{输出费用：} 0.2\text{M tokens} \times \$0.28 = \$0.056$$
$$\text{单日总开销} = \$0.266 \text{ (每月约合人民币 58 元)}$$

*高达 95% 的成本降幅，解释了为什么在架构设计时，选择高性价比的模型能直接决定一个 AI 项目能否商业落地。*

## 🖥️ 3. 三大托管模式的经济学对比

如果你决定使用 Llama 3 或 Qwen 等开源模型，自己买显卡、租显卡和买 API 密钥相比，哪种更划算？

| 托管模式 | 资金特征 | 最佳适用场景 |
| :--- | :--- | :--- |
| **商业按量 API** | 变动成本。有用户访问才扣钱，没用户访问一分钱不用花。 | **项目初期 / 低并发**：如果每天只有几十个人用，直接买 Key 是最省钱的。 |
| **云端 GPU 租用** | 固定时间成本（如租用 A100 大约 1.5美元/小时）。 | **高并发 / 稳定访问**：如果有上万人不间断使用，租一整张显卡跑 vLLM 的总价会远低于按 token 付费的 API。 |
| **自购本地显卡** | 高额一次性固定资产投入（如买 RTX 4090 需上万元）。 | **离线开发 / 极高隐私**：买断后运行完全免费，且数据百分之百不出门。 |
