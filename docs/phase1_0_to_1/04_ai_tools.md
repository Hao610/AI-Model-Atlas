# AI Tools Guide 🛠️

> 📅 Last updated: 2026-06. AI ecosystems iterate rapidly; please refer to official documentation for the latest versions and pricing.

[English] | [中文 (04_ai_tools_zh.md)](04_ai_tools_zh.md)

Before diving into APIs and code, you should understand the existing landscape of user-ready tools. This guide categorizes the most popular AI tools you can use on your browser or desktop right now.

---

## 💬 1. General & Chat Assistants

These are your primary conversers, researchers, and writers.

| Tool | Creator | Best For | Standout Feature |
| :--- | :--- | :--- | :--- |
| **ChatGPT** | OpenAI | General reasoning, brainstorming, writing. | **GPT-4o / o1 / o3** models; Voice Mode; Canvas workspace. |
| **Claude** | Anthropic | Coding, complex analysis, writing polish. | **Sonnet** model; "Artifacts" panel to run code/previews live. |
| **DeepSeek Chat** | DeepSeek | Math, logic, high-value coding at low cost. | Excellent reasoning (**R1**), deeply optimized for developer use. |
| **Gemini** | Google | Integration with Google ecosystem, YouTube. | Massive **2 Million Token** context window. |

---

## 🎨 2. Visual & Image Generation

No code needed—describe an image, get high-quality art.

* **Midjourney**: Best for cinematic, artistic, and highly detailed images. Runs inside Discord.
* **DALL-E 3**: Best for understanding text inside images. Built directly into ChatGPT Plus.
* **Stable Diffusion / Flux**: Outstanding open-source image models. You can run them on your own computer (using tools like ComfyUI) to generate uncensored art for free.

---

## 💻 3. Coding Assistants

These live inside your code editor (like VS Code) to write code with you.

* **GitHub Copilot**: Autocompletes your code line-by-line.
* **Cursor**: A modified VS Code editor designed specifically for AI. It can read your entire workspace, edit multiple files at once, and build web apps from simple English commands.
* **Windsurf**: Another AI-native IDE with a focus on collaborative agentic coding.

---

## 🔄 4. Low-Code Workflows & Agents

When chat prompts aren't enough, and you want to build automated pipelines.

* **Coze / Dify**: Visually drag-and-drop boxes to build AI chatbots with memory, Google search capability, and database storage.
* **Make / Zapier**: Connect AI to your email, spreadsheet, and calendar to automate daily tasks.
