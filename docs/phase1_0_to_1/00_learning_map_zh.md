```text
               ┌────────────────────────┐

               └───────────┬────────────┘

   【1. 应用体验路径】 【2. 低代码搭建路径】 【3. 硬核研发路径】
      (零代码提效)       (搭建知识库/Agent)    (代码/微调/部署)
```

### 路径 1：零代码应用体验路径 (User Path)
* **适合人群**：文案策划、项目经理、设计师，以及希望在日常工作中使用 AI 工具提高生产力、拒绝写代码的同学。
* **通关路线**：
  * **第一阶段 (全修)**：从《什么是 AI？》一直读到《核心词汇表》。重点掌握 **ROLE** 提示词框架，并搞懂开源模型与商业授权的红线。
  * **第二阶段 (选修)**：阅读《无代码 Agent 搭建》，学会用 Dify 可视化配置自己的定制机器人。

### 路径 2：低代码系统搭建路径 (Builder Path)
* **适合人群**：创业者、IT 专员、产品经理。希望为企业内部搭建私有文档检索知识库（RAG）或复杂的自动化业务工作流。
* **通关路线**：
  * **第一阶段 (全修)**：阅读基本概念。
  * **第二阶段 (全修)**：深度攻克 RAG、向量数据库、多模态 AI 以及 AI 工作流架构。这是你设计企业方案的骨架。
  * **第三阶段 (选修)**：学习《本地大模型运行》，学会在自己的电脑上用 LM Studio 离线启动模型。

### 路径 3：硬核 AI 研发路径 (Engineer Path)
* **适合人群**：软件开发工程师、数据科学家。需要手写 Python 代码，调用接口，开发前端聊天网页，在私有数据集上微调模型，并把模型部署到云端显卡服务器供千万人使用。
* **通关路线**：
  * **全栈通关 (1 至 32 模块)**：按顺序从头读到尾。重点攻克 Python API 调用、向量 Embedding 数理匹配、智能体开发框架 (LangGraph)、LoRA 微调实操、显卡选型以及云端 vLLM 高并发部署。

## 💡 教授的阅读建议

* **字典式查阅**：你不需要一天内把它读完。把它当作一本“AI 开发字典”。当你在开发或开会时听到“Embedding”或“量化”等名词，随时跳回对应的章节查阅即可。
* **比喻先行**：每一章的开头都有我精心设计的“生活比喻”。在看 Python 代码或命令行之前，先看懂比喻，在大脑里建立起直观的物理图像，能让你的学习效率翻倍。
