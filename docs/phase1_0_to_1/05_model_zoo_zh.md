# 模型动物园一览 🦁

[[English] (05_model_zoo.md)](05_model_zoo.md) | [中文]

在 AI 开源社区，大家亲切地把各种预训练模型的集合称为 **“模型动物园 (Model Zoo)”**。由于开发大模型的机构和公司层出不穷，很多新手面对各种名字容易一头雾水。

本章为你提供一个快速查阅字典，对比目前最主流的几大**大语言模型 (LLM) 家族**。

## 💡 概念澄清：闭源商业 API vs. 开放权重开源模型

* **闭源模型 (Proprietary / Closed Source)**：模型的所有权和文件都在发布公司手中。你只能通过网络向他们付费发起请求（使用 API 密钥），你无法下载模型文件，也无法探知其内部结构。
* **开放权重模型 (Open Weights / Open Source)**：发布方把训练好的参数文件公开允许下载。你可以把整个模型搬到自己的电脑上运行，数据不用传到网上，也可以用自己的数据去修改它，自由度极高。
