# 常用 AI 工具全景图 🛠️

[[English] (04_ai_tools.md)](04_ai_tools.md) | [中文]

在开始写代码或调用接口之前，我们应该先了解有哪些现成的、开箱即用的 AI 工具。这篇指南会为你梳理目前最主流、最实用的 AI 应用，并按使用场景分类。

---

## 💬 1. 综合聊天与文字助手

它们是你的日常对话者、研究助理和撰稿人。

| 工具名称 | 出品公司 | 擅长领域 | 亮点功能 |
| :--- | :--- | :--- | :--- |
| **ChatGPT** | OpenAI | 通用逻辑推理、头脑风暴、文案起草 | 搭载 **GPT-4o/o1/o3**；支持语音对话；有 Canvas 画布模式 |
| **Claude** | Anthropic | 代码编写、复杂长文本分析、文学润色 | 搭载 **Sonnet**；有 Artifacts 预览面板，可直接运行网页代码 |
| **DeepSeek** | 深度求索 | 数理逻辑、极高性价比的代码编写 | 极强深度思考能力 (**R1**)，目前备受开发者青睐 |
| **Gemini** | 谷歌 | 与谷歌生态结合（邮件、YouTube、文档） | 拥有夸张的 **200 万 Token** 超大上下文窗口 |

---

## 🎨 2. 图像与视觉生成

不需要懂画画——用文字描述画面，就能得到高质量图片。

* **Midjourney**：生成电影级、艺术感极强的原画。目前主要在 Discord 社群内运行。
* **DALL-E 3**：能非常完美地理解画面中的文字（不会把英文字母画错）。直接集成在 ChatGPT Plus 中。
* **Stable Diffusion / Flux**：优秀的开源生图模型。你可以通过 ComfyUI 等工具把它们安装在本地电脑上，免费、本地化地生成没有任何限制的图片。

---

## 💻 3. 编程辅助 (AI 程序员)

它们直接嵌入你的代码编辑器（如 VS Code）里，和你一起“结对编程”。

* **GitHub Copilot**：老牌代码补全插件，能预测你下一行要写的代码。
* **Cursor**：专为 AI 打造的 VS Code 深度定制版编辑器。它能阅读你整个文件夹，一次性修改多个文件，并用纯中文命令帮你从零搭起网页。
* **Windsurf**：另一款主打“智能代理协作”的 AI 原生编辑器，智能化程度极高。

---

## 🔄 4. 低代码工作流与 Agent (智能体)

当你觉得单一的聊天框解决不了复杂的自动化流程时。

* **Coze (扣子) / Dify**：可视化“拖拉拽”平台。你可以像连线拼图一样，给大模型接上记忆库、搜索引擎和数据库，拼装出一个专属的 AI 员工。
* **Make / Zapier**：把 AI 连接到你的邮箱、表格和日程表，自动替你回复客户邮件。
