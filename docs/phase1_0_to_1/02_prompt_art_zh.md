# 提示词艺术 🎨

[[English] (02_prompt_art.md)](02_prompt_art.md) | [中文]

跟 AI 聊天和在谷歌上搜索是完全不同的。谷歌看重的是“关键词”，而跟 AI 交流，你更像是在**执导一个聪明的助理**。

你给 AI 发送的指令文字就叫做**提示词 (Prompt)**，而研究如何写出高效提示词的学问，就是**提示词工程 (Prompt Engineering)**。

---

## 🏛️ R.O.L.E. 提示词黄金框架

如果你问得模糊，AI 就会回答得平庸。想要获得高质量的回答，可以使用 **ROLE** 公式：

| 要素 | 说明 | 示例 |
| :--- | :--- | :--- |
| **R**ole (角色/是谁？) | 给 AI 设定一个明确的职业身份或个性。 | *“你是一位资深的雅思英语老师。”* |
| **O**bjective (目标/做什么？) | 明确说明你需要 AI 帮你完成什么任务。 | *“请帮我解释 affect 和 effect 的区别。”* |
| **L**imitations (限制/怎么做？) | 设定边界条件，如字数、语气、禁止使用的词汇。 | *“用大白话解释，字数控制在100字以内，不要用深奥的语法术语。”* |
| **E**xample (示例/格式？) | 给 AI 喂一个你想要的输出样式，让它照猫画虎。 | *提供一个你期望的排版格式样例。* |

## 🎯 糟糕提示词 vs. 优秀提示词对比

让我们看一个实际场景的差别：

### 场景：写一封向老板请假生病的邮件

| 糟糕的提示词 | 优秀的提示词 (使用 ROLE 框架) |
| :--- | :--- |
| *帮我写封请假信给老板，说我病了。* | *你是一位严谨的公司行政助理 (Role)。请帮我写一封向老板请假的邮件，说明我因突发高烧今天无法上班 (Objective)。语气要礼貌、得体且简练 (Limitations)。请按以下格式输出：邮件主题、称呼、正文、落款 (Example)。* |
| **效果**：生成一段非常口语化、甚至有些随意的文字，可能不符合你们公司的企业文化。 | **效果**：生成一封格式严谨、措辞极为得体的专业邮件，你只需要复制粘贴即可。 |

## ⚡ 核心提效技巧

### 1. 零样本 (Zero-Shot) 与 少样本 (Few-Shot) 提示
* **零样本 (Zero-Shot)**：直接让 AI 执行任务，不给任何参考例子。
  * *示例*：*“把这句话翻译成日语：你好，今天天气真好。”*
* **少样本 (Few-Shot)**：在提问前，先给 AI 演示 1~2 个正确的“问题-答案”配对。这在需要固定输出格式或模仿某种文风时**极其强大**。
  * *示例*：
    ```text
    输入：开心 -> 输出：😊
    输入：难过 -> 输出：😢
    输入：激动 -> 输出：
    ```
    *(AI 会瞬间心领神会，输出 😆)*

### 2. 思维链 (Chain of Thought, 简称 CoT)
当你让大模型做复杂的数学题、脑筋急转弯或逻辑推理时，它有时会因为“脱口而出”而算错。解决这个问题的魔法咒语是：
> *“请一步一步地思考，并给出推理过程。”*

这会强制模型在给出最终答案之前，先在心里“打草稿”写出逻辑步骤，能显著提升逻辑题的准确率。
