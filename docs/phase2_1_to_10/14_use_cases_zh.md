# 真实应用案例 💼

[[English] (14_use_cases.md)](14_use_cases.md) | [中文]

让我们来看一看，工业界是如何将 **Agent + RAG + LLM** 的工作流组合起来，去解决真实的商业痛点的。以下是目前最普遍、最受企业欢迎的六个 AI 项目配置模板。

---

## 🤖 1. 自动客服助理

* **目标**：24小时不间断解答用户的重复性咨询，无需人工值守。
* **系统架构**：
  * **关联知识库**：公司产品 FAQ 常见问题、退换货条款、快递配送时效。
  * **绑定工具**：快递单号查询 API、人工客服转接插件（当用户执意要求人工时，向 Slack/钉钉 发送提醒）。
  * **核心提示词**：*“你是一位亲切的线上网店客服。你的回答必须保持专业礼貌，且仅能基于知识库提供的事实进行作答。严禁猜测用户的快递单号，必须调用快递查询工具获取真实状态。”*

---

## 📂 2. 企业内部知识库

* **目标**：帮助新老员工快速在成百上千页的技术规范、HR规章和法务合同中检索信息。
* **系统架构**：
  * **关联知识库**：公司几十本 HR 规章手册、技术架构 PDF 集合。
  * **底层数据库**：使用 PostgreSQL 并启用 `pgvector` 扩展插件。
  * **核心提示词**：*“你是一位企业内部行政助手。在回答员工问题时，必须在回答末尾注明出处文件和章节名（例如：『根据《HR手册2026版》第4.2章...』）。”*

---

## ✍️ 3. AI 写作助手

* **目标**：辅助市场公关、自媒体作者快速起草符合品牌调性的邮件、文案和社交广告。
* **系统架构**：
  * **关联知识库**：公司的品牌调性手册、过往爆款文案示例（用作 Few-Shot 样本）。
  * **工作流设计**：多步骤画布。步骤一生成文章大纲，步骤二填充正文，步骤三由另一个“扮演主编的角色”对文章遣词造句和敏感词进行过滤和润色。
  * **核心提示词**：*“你是一位资深文案策划。文字风格要简练、富有张力。严禁使用『颠覆性』、『赋能』、『闭环』等陈腐的互联网黑话。”*

---

## 🌍 4. 专业行业翻译官

* **目标**：超越传统翻译软件的生硬“直译”，能够准确理解行业黑话、俚语和特定上下文。
* **系统架构**：
  * **关联知识库**：行业术语中英文对照表（如医学专业术语、游戏开发黑话）。
  * **工作流设计**：提供对照例句，让 AI 学习“信达雅”的翻译逻辑。
  * **核心提示词**：*“请将以下软件开发文档从英文翻译为中文。注意，当上下文处于 Docker 容器技术时，单词 'container' 应翻译为『容器』，而绝不能翻译为『货柜』。”*

---

## 📊 5. AI 数据分析师

* **目标**：让不懂 SQL 代码的运营经理能够用大白话向数据库提问，自动生成分析图表。
* **系统架构**：
  * **关联知识库**：数据库的表结构清单（表名、各列数据定义、表与表之间的关联关系）。
  * **绑定工具**：Python 代码运行沙箱（Sandbox）。
  * **工作流设计**：
    1. 运营经理输入纯中文问题。
    2. 大模型自动编写出 SQL 查询语句或 Pandas 计算脚本。
    3. 平台在安全沙箱里执行这段代码，拉取数据。
    4. 大模型阅读拉取出的数据，将其规整为美观的 Markdown 表格或折线图输出。
