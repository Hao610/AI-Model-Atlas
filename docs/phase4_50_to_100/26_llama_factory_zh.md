# LLaMA-Factory 训练指南 🛠️

[[English] (26_llama_factory.md)](26_llama_factory.md) | [中文]

手写 PyTorch 训练循环、处理硬件分布式通信、配置 Tokenizer 切词算法对于零基础开发者来说非常困难。

为了极大降低微调的技术门槛，开源社区推出了 **LLaMA-Factory**。它是一个一站式的模型训练控制台，提供了一个极为美观的网页可视化界面，让你只需通过鼠标“点选”，就能完成大模型的全部训练配置与启动。

---

## 📅 第一步：准备微调训练数据集

LLaMA-Factory 默认支持最通用的 **Alpaca 格式**（一个包含“指令-输入-输出”对的 JSON 列表文件）。

新建一个名为 `my_data.json` 的文件，写入你的训练样本：

```json
[
  {
    "instruction": "请把以下患者的俗语翻译为专业医学术语。",
    "input": "我嗓子有点干，身上觉得发烫。",
    "output": "患者自诉轻度咽部充血，伴随低热症状。"
  },
  {
    "instruction": "请把以下患者的俗语翻译为专业医学术语。",
    "input": "我肚子疼得拧成了麻花。",
    "output": "患者自诉伴有急性腹部痉挛性疼痛。"
  }
]
```

* **instruction (指令)**：你希望用户对 AI 发送的命令。
* **input (输入)**：(可选) 额外的上下文背景或查询材料。
* **output (输出)**：你希望 AI 模仿和学习的百分之百正确的标准答案。

---

## 🚀 第二步：安装 LLaMA-Factory 框架

打开你的命令行终端，依次执行以下命令：

```bash
git clone https://github.com/hiyouga/LLaMA-Factory.git
cd LLaMA-Factory
pip install -e .[metrics,bitsandbytes]
```

---

## 🖥️ 第三步：一键启动 Web UI 可视化控制台

在终端中输入以下命令启动网页控制面板：

```bash
llamafactory-cli webui
```

打开浏览器，访问 `http://localhost:7860`。你将看到功能完备的 LLaMA-Factory 中文操作面板：

```text
┌────────────────────────────────────────────────────────┐
│ LLaMA-Factory 可视化微调面板                             │
├────────────────────────────────────────────────────────┤
│ 模型名称：    [ Llama-3-8B-Instruct ]                  │
│ 模型路径：    [ /你的本地模型文件夹/llama-3-8b ]         │
│ 微调阶段：    [ Supervised Fine-Tuning (SFT) ]         │
│ 微调方法：    [ LoRA ]                                 │
├────────────────────────────────────────────────────────┤
│ 数据集列表：  [ my_data.json ]                         │
│ 学习率：      [ 2e-4 ]       训练轮数 (Epoch): [ 3.0 ]  │
│ LoRA 秩(r)：  [ 8 ]          LoRA 投影(alpha): [ 16 ]  │
├────────────────────────────────────────────────────────┤
│ [ 开始训练 ]  [ 预览命令行 ]                             │
└────────────────────────────────────────────────────────┘
```

---

## ⚙️ 第四步：核心参数配置指南

在网页上配置以下核心指标：

1. **模型名称 / 模型路径**：选择你要微调的底座模型（如 Llama 3 或 Qwen 2.5）。
2. **微调阶段**：选择 **Supervised Fine-Tuning (SFT，监督微调)**。
3. **微调方法**：选择 **LoRA**。
4. **数据集**：在下拉菜单里勾选你准备好的 `my_data.json`。
5. **超参数配置**：
   * **学习率 (Learning Rate)**：设为 `2e-4`（默认黄金值）。
   * **训练轮数 (Epoch)**：设为 `3`（代表让模型把你的数据集从头到尾读完并学习 3 遍）。
   * **LoRA 秩 (Rank)**：设为 `8` 或 `16`。
6. **输出目录**：填写你希望保存最终微调生成的 `10MB` 左右适配器权重文件的路径。

一切就绪后，滑到页面最底部，点击 **【开始训练】**。控制台会实时绘制出 Loss 损失值折线图（折线平稳下滑代表模型学得很好）和训练进度条。当进度条达到 100% 时，你的专属 LoRA 适配器就大功告成了！
