# 🚀 快速上手：1 分钟通关指南

欢迎使用 **混合动力认知 RAG 系统**！请跟随以下简单的 5 步指南，快速拉起并体验本系统。

---

## 🏃‍♂️ 第一步：安装系统依赖
确保您的系统已安装 Python 3.9 或更高版本，然后执行：
```bash
pip install -r requirements.txt
```

---

## 💻 第二步：启动本地模型（默认离线运行模式）
如果您希望在本地完全免费地运行大模型，请确保后台已开启 Ollama 客户端，并拉取 Llama 3 模型：
```bash
ollama pull llama3
```
*(注：您也可以在启动系统后，在左侧侧边栏中随时热切换为云端 API 模式，并填入您的 OpenAI 或 DeepSeek API Key)*。

---

## 🌐 第三步：一键拉起应用
在当前目录下启动 Streamlit 网页控制台：
```bash
python app.py
```
程序启动后会自动在您的浏览器中打开前端交互页面（默认地址为 `http://localhost:8501`）。

---

## 📂 第四步：上传并解析知识库文档
1. 找到右侧面板的 **"📂 Document Context Ingestion"** 区域。
2. 拖拽或上传任意一个 PDF 文件。
3. 系统会自动对其进行智能段落切片、生成向量 Embedding 并索引存入本地 ChromaDB 向量库。

---

## 💬 第五步：体验智能检索与性能面板
1. 在聊天框中输入一个关于您上传的文档的问题，点击发送。
2. **观察响应时延**：观察右侧 `System Controller Traces` 区域，查看本次生成的首 Token 延迟 (TTFT) 与 Token 输出速度指标。
3. **触发语义缓存秒回**：再次发送相同的问题（或语义非常相近的问题），您会发现系统通过 **语义缓存 (Semantic Cache)** 实现了**瞬时秒回 (0.00s)**，完全绕过了 RAG 和大模型生成链路。
4. **动态微调参数**：在左侧侧边栏中，您可以一键开启/关闭重排 (Rerank) 或改写 (Rewrite)，并滑动微调相似度截断阈值，实时观测 RAG 检索上下文的质变。
