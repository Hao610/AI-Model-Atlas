# 安全政策

感谢您帮助维护 AI-Model-Atlas 的安全性！

## 受支持的版本

目前，我们仅为 `main` 分支上的最新代码提供安全更新支持。

| 版本 | 是否支持安全更新 |
| ------- | ------------------ |
| `main`  | :white_check_mark: (支持) |
| forks (衍生版本) | :x: (不支持) |

## 报告漏洞

请 **不要** 通过公开的 GitHub Issues 来报告安全漏洞。

如果您认为在 AI-Model-Atlas 中发现了安全漏洞（例如：意外暴露的 API 密钥、RAG 实战代码中的提示词注入风险，或第三方依赖库的高危漏洞），请直接通过电子邮件私下联系主理人：

**邮箱：** loichianghao@gmail.com

**报告中请包含：**
* 漏洞的详细描述。
* 重现该漏洞的步骤。
* （可选）潜在的缓解措施或修复建议。

我们非常重视所有安全报告，并会努力在 48 小时内确认您的报告。我们将与您密切合作，验证问题并尽快发布修复补丁。
